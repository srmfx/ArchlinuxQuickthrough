# ArchlinuxQuickthrough
This is a quickthrough for archlinux/linux troubleshooting

I mostly started this when I was still at University as a way of keeping things I have studied noted and written down, which I have decided to share after years and years of keeping it to myself.

Most of the stuff I've wrote is about Linux, Archlinux, Nvidia and Intel on archlinux, which is the main stuff I've been using throughout those years.

You can find the text file on the Documents directory, it's meant to be open, visualized and queried with 'nvim', although you may use any other text editor you want.
